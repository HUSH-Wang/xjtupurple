### 如何使用
1. 打开微信;
2. 将这个网址[http://xjtupurple.codes](http://xjtupurple.codes)发送给一个人;
3. 点击发送出去的链接;
4. 点击"返校申请";
5. 将内容修改为自己的信息;
6. 上传照片, 选择生成码颜色;
7. 点击"交大校园码"
8. 生成紫码界面.

### 多终端共享
